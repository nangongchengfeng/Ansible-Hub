## 阶段一：项目初始化与基础框架

### 任务 1：后端项目初始化
**类型**：AFK
**用户故事**：-

**内容**：

- 配置 SQLAlchemy（异步引擎）
- 配置 Alembic 数据库迁移
- 配置 Redis 连接
- 配置 Celery（Worker + Beat）
- 配置 Fernet 加密
- 配置环境变量管理（pydantic-settings）
- 配置日志系统
- 配置 CORS
- 配置项目依赖（requirements.txt）
- 配置代码规范（Black + isort + flake8）

**验收标准**：
- [ ] 项目能正常启动

- [ ] Celery 能正常启动
