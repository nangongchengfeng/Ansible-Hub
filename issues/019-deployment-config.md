# 部署配置

## What to build

实现项目的部署配置，包括 Dockerfile、docker-compose.yml、生产环境配置文档、环境变量示例、数据库初始化脚本。

## Acceptance criteria

- [ ] Dockerfile（后端）编写完成
- [ ] Dockerfile（Celery Worker）编写完成
- [ ] Dockerfile（Celery Beat）编写完成
- [ ] docker-compose.yml 编写完成（包含 MySQL + Redis）
- [ ] 生产环境配置文档编写完成
- [ ] 环境变量配置示例（.env.example）编写完成
- [ ] 数据库初始化脚本编写完成
- [ ] Docker 镜像能正常构建
- [ ] docker-compose 能正常启动完整环境
